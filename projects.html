Data Ingestion Pipeline Stress Testcode
Data Ingestion Pipeline Stress Testclose

Developed a tool used across the project44 engineering teams to test the Apache Kafka data ingestion pipeline used for reception of incoming customer data. This REST API lived in our Integration stack, and allowed users to send a simply formatted GET request and trigger load testing of the pipeline. Generated load and utilized several concepts such as: reservoir sampling for random access of realistic information, asynchronous REST calls to external and internal systems' APIs, depth-first-searching of a self-made weighted graph structure and continuous integration via deployment through Jenkins, into a Kubernetes pod, in a Docker container.

Ocean Shipment Mock Data Providercode
Ocean Shipment Mock Data Providerclose

In order to increase productivity and smoke-testing of the Ocean Shipment managing and tracking services of project44's technologies, I created a tool that could generate realistic, albeit false, data, to allow our polling services to hit a false endpoint and poll false data as opposed to polling real data for testing purposes. This took the form of a simple REST API that, upon being hit with a GET request from our polling services would return data that mocked the exact data we received from external data sources. This made our testing processes for our ocean services entirely in house, and took away our dependency on external services.

Cucumber Slack Notifiercode
Cucumber Slack Notifierclose

Cucumber is a test suite designed for ease of use across teams including non-engineers. This test suite utilizes naturally written test methods, such as "When this is true, do this," to initialize test methods written in Java. At project44, the test suites were written in Cucumber features and as such, it benefitted us to have a notification of test success for our continous testing through Jenkins. This notifier had been in development previously, but remained untouched and unfunctioning, so I took the existing source and developed it for internal use.
